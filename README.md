# 🏦 Bank Loan Analysis Dashboard (Power BI)

##  Project Overview

This Power BI dashboard provides a comprehensive analysis of bank loan data, helping to understand customer behavior, loan performance, and risk factors.

The dashboard enables financial institutions to make data-driven decisions by analyzing loan applications, approvals, repayments, and defaults.

---

##  Objective

* Analyze loan distribution and performance
* Identify high-risk customers
* Monitor repayment trends
* Support better lending decisions

---

##  Dataset Description

The dataset includes:

* Loan applications
* Customer financial details
* Credit history
* Loan status (Approved / Rejected / Default)
* Repayment and disbursement data

---

##  Key Insights & Features

*  Total Loan Applications Overview
*  Total Funded Amount & Received Amount
*  Monthly Trends Analysis
*  Default Loan Analysis
*  Customer Segmentation
*  Loan Status Breakdown (Good vs Bad Loans)

---

## 📊 Dashboard Visuals

* KPI Cards (Total Loans, Funded Amount, Received Amount)
* Bar Charts (Loan Status, Purpose, Grade)
* Line Charts (Monthly Trends)
* Pie/Donut Charts (Loan Distribution)
* Tables for detailed insights

---

##  Domain Knowledge

Bank loan analysis is important for:

* Risk Assessment
* Fraud Detection
* Creditworthiness Evaluation
* Portfolio Management
* Profitability Analysis

The loan process includes application, verification, approval, disbursement, and repayment tracking.
